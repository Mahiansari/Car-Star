CAR STAR – Your Ultimate Car Platform
CAR STAR is a fully responsive front-end website designed for car enthusiasts and potential buyers. The platform provides detailed information about world-famous cars, including specifications, images, and key features. Built with HTML, CSS, and JavaScript, it offers a clean and user-friendly experience across all devices.

⚠️ Note: This is a simple static website with no backend functionality. All data and interactions are front-end only.

Features:
Home Page – Welcoming hero section with platform introduction
Brand Listings – Showcase of world-famous cars including:
Gold Fantasy (Lamborghini)
Rolls Royce
Bugatti Chiron Super Sport
Ferrari 488 Pista Spider
McLaren 720S Spider
Mercedes Benz S-Class
BMW M8 (Manhart's Creation)
Audi R8 Spyder

Search Bar – Interface for searching cars (UI ready)
Contact Section – Address, phone, email, and inquiry options
Side Map – Location reference
Newsletter Subscription – Stay updated with latest cars
Fully Responsive – Works smoothly on mobile, tablet, and desktop
Modern UI – Clean layout with organized sections

Technologies Used:
HTML5 – Structure and content
CSS3 – Styling and layout
JavaScript – Interactivity (future enhancements)
